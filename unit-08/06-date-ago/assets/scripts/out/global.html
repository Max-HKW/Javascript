getDayAgo(date, days) → {number}

Returns the day of the month 'n' days ago or ahead from the given date. This function handles both positive and negative 'days' parameters.
Parameters:
Name Type Description
date Date The starting date to subtract or add days.
days number The number of days to add (positive) or subtract (negative).
Source:
  • main.js, line 22
Returns:
- The day of the month after adding or subtracting the specified number of days.
Type
number
Example
const today = new Date('2025-03-30');
console.log(getDayAgo(today, -7));  // 23 March 2025 (7 days ago)
console.log(getDayAgo(today, 7));   // 6 April 2025 (7 days ahead)
