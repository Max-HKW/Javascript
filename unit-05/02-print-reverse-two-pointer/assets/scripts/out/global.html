printReverse(string) → {string}

Reverses a given string using the two-pointer approach. The method works by initializing two pointers: one at the beginning of the string and one at the end. These pointers move towards each other while swapping the characters at their respective positions until they meet or cross. This technique provides an efficient way to reverse a string in-place with O(n) time complexity and O(1) space complexity.
